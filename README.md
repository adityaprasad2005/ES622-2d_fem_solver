# 2D Finite Element Method (FEM) Solver for Plane Stress Analysis

![MATLAB](https://img.shields.io/badge/MATLAB-R2023a+-blue.svg)
![Gmsh](https://img.shields.io/badge/Gmsh-Open%20Source%20Meshing-brightgreen.svg)

A custom, high-performance Finite Element Method (FEM) solver built entirely in MATLAB. This project analyzes the **plane stress distribution** and **stress concentration** in a finite rectangular plate containing a central circular hole subjected to uniaxial tension. 

Developed as part of the ES622 curriculum at the Indian Institute of Technology (IIT) Gandhinagar.

---

## 📖 Project Overview
Stress concentration is a critical phenomenon in solid mechanics. While theoretical solutions (like Kirsch's equations) exist for *infinite* plates, finite plates require numerical approximations. 

This solver bridges the gap between theoretical continuum mechanics and computational engineering by:
1. **Parsing** unstructured mesh data generated by Gmsh.
2. **Assembling** a global stiffness matrix using 4-node isoparametric quadrilateral (Q4) elements and Gauss-Legendre numerical integration.
3. **Enforcing** quarter-symmetry kinematic boundary conditions via the Penalty Method.
4. **Solving** the linear system and recovering localized Von Mises stress fields.

## ✨ Key Features
* **Custom Mesh Parser:** Reads natively exported `.msh` (Version 2 ASCII) files from Gmsh.
* **Isoparametric Q4 Elements:** Full implementation of bilinear shape functions, Jacobian mapping, and strain-displacement ($B$) matrices.
* **Automated Post-Processing:** Automatically computes elemental strains, stresses at the geometric centroids, and generates scaled deformation plots.
* **Rigorous Grid Convergence:** A standalone parametric study script that automatically evaluates local stress, global strain energy, and computational cost across varying mesh densities.
